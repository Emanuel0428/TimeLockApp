2. Descripción del Servicio

TimeLock es una aplicación móvil para Android diseñada para ayudar a los usuarios a gestionar su tiempo de pantalla mediante:

  • Monitoreo y seguimiento de métricas de uso del dispositivo
  • Sistema de tokens ganados por sesiones de enfoque y desafíos completados
  • Desafíos diarios personalizables para mejorar hábitos digitales
  • Modo enfoque Pomodoro con etiquetas y estadísticas de sesiones
  • Bloqueo de aplicaciones y sitios web seleccionados
  • Detección de uso en movimiento vs. estacionario

4. Permisos y Acceso al Dispositivo

Para funcionar correctamente, TimeLock requiere ciertos permisos en su dispositivo:

  • Estadísticas de Uso (UsageStats): Para monitorear el tiempo de uso de aplicaciones
  • Superposición de Pantalla (Overlay): Para mostrar bloqueos sobre aplicaciones específicas
  • Ubicación: Para detección de movimiento (caminando vs. estacionario)
  • Notificaciones: Para enviar recordatorios y alertas de desafíos

Todos los datos recopilados se almacenan localmente en su dispositivo y no se transmiten a servidores externos.
