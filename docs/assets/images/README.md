# Carpeta de Imágenes

Esta carpeta contiene las imágenes generales utilizadas en el landing page.

## Contenido sugerido:

- Logo de la aplicación
- Iconos personalizados
- Imágenes decorativas
- Fondos

## Formato recomendado:

- **Logos**: SVG o PNG (fondo transparente)
- **Iconos**: SVG (preferiblemente)
- **Imágenes**: WebP o PNG optimizadas
- **Peso máximo**: 200KB por imagen

## Optimización:

Usa herramientas como:
- [TinyPNG](https://tinypng.com/) para PNG/JPG
- [SVGOMG](https://jakearchibald.github.io/svgomg/) para SVG
- [Squoosh](https://squoosh.app/) para WebP
