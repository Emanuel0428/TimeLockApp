# Carpeta de Screenshots

Esta carpeta contiene las capturas de pantalla de la aplicación TimeLock.

## Contenido:

Coloca aquí los screenshots de las diferentes pantallas de la app:

- `home.png` - Pantalla principal
- `metrics.png` - Dashboard de métricas
- `focus.png` - Modo enfoque Pomodoro
- `tokens.png` - Sistema de tokens
- `challenges.png` - Desafíos
- `settings.png` - Configuración

## Especificaciones:

- **Resolución**: 1080x2400px (ratio 9:16) o similar
- **Formato**: PNG o WebP
- **Orientación**: Vertical (portrait)
- **Peso máximo**: 300KB por imagen
- **Optimización**: Comprimir con TinyPNG o Squoosh

## Uso:

Reemplaza el placeholder en `index.html` sección hero:

```html
<div class="phone-screen">
    <img src="assets/screenshots/home.png" alt="TimeLock App - Home">
</div>
```

## Tips:

- Usa fondos consistentes (mismo tema oscuro de la app)
- Captura con datos realistas (no uses "Lorem ipsum")
- Asegúrate de ocultar información sensible si la hay
- Considera crear un carrusel para mostrar múltiples pantallas
