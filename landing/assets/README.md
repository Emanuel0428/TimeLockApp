# Assets - Recursos del Landing Page

Esta carpeta contiene todos los recursos estáticos utilizados en el landing page de TimeLock.

## 📁 Estructura

```
assets/
├── images/         # Imágenes generales (logos, decoraciones)
├── screenshots/    # Capturas de pantalla de la app
└── icons/          # Iconos y favicons
```

## 🎨 Guía de Uso

### Images (images/)
Contiene imágenes generales como:
- Logo de TimeLock
- Imágenes decorativas
- Fondos
- Iconos personalizados

### Screenshots (screenshots/)
Capturas de pantalla de la aplicación para el hero section y galerías:
- Formato: PNG o WebP
- Resolución: 1080x2400px (móvil vertical)
- Peso máximo: 300KB

### Icons (icons/)
Favicons y iconos de aplicación:
- Favicon en múltiples tamaños
- Apple touch icons
- Android chrome icons
- Manifest para PWA

## 🔧 Optimización

Antes de agregar imágenes a estas carpetas, optimízalas usando:

- **PNG/JPG**: [TinyPNG](https://tinypng.com/)
- **WebP**: [Squoosh](https://squoosh.app/)
- **SVG**: [SVGOMG](https://jakearchibald.github.io/svgomg/)

## 📝 Nomenclatura

Usa nombres descriptivos en minúsculas con guiones:

✅ Correcto:
- `logo-timelock.svg`
- `screenshot-home.png`
- `icon-play-store.png`

❌ Incorrecto:
- `Logo.svg`
- `Screenshot 1.png`
- `icon_playstore.png`

## 💡 Tips

1. Mantén los archivos organizados en sus respectivas subcarpetas
2. No subas imágenes sin optimizar
3. Usa formatos modernos (WebP) cuando sea posible
4. Mantén consistencia en las dimensiones de screenshots
5. Documenta cualquier imagen especial en el README de su carpeta
